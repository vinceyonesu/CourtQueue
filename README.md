# CourtQueue 🎾🏓

A web app for tennis and pickleball players to manage court wait times efficiently.

## Features
- Players join a digital queue
- Live queue display with positions
- Timer for playtime
- Estimated wait time for players in queue

## Tech Stack
- Front-end: React.js
- Back-end: Node.js + Express
- Real-time: Socket.io
- Database: PostgreSQL
- Testing: Jest
